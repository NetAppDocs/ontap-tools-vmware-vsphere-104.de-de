= Aktualisieren Sie die Anmeldeinformationen des vCenter Servers.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Anmeldeinformationen der vCenter Server-Instanz über die Wartungskonsole aktualisieren.

.Bevor Sie beginnen
Sie benötigen die Anmeldedaten eines Wartungsbenutzers.

*Über diese Aufgabe*

Wenn Sie die Anmeldeinformationen für vCenter Server nach der Bereitstellung geändert haben, müssen Sie die Anmeldeinformationen mit diesem Verfahren aktualisieren.

.Schritte
. Öffnen Sie vom vCenter Server aus eine Konsole für den VASA-Anbieter.
. Melden Sie sich als Wartungsbenutzer an.
. Eingeben `2` , um das Systemkonfigurationsmenü auszuwählen.
. Eingeben `8` um die vCenter-Anmeldeinformationen zu ändern.
